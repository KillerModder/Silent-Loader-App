# Silent Loader

Silent Loader é um aplicativo Android desenvolvido para download e gerenciamento de configurações online de forma simples, rápida e leve.

## Recursos

- Configurações online
- Interface moderna
- Sistema leve e rápido
- Preview de imagens
- Links de download
- Tutoriais integrados
- Atualizações via Firebase

---

## Download

Baixe a versão mais recente na aba [⬇ Releases](https://github.com/KillerModder/Silent-Loader-App/releases).

---

## Requisitos

- Android 7 ou superior
- Conexão com internet

---

## Instalação

1. Baixe o APK na aba Releases
2. Permita instalação de fontes desconhecidas
3. Instale normalmente
4. Abra o aplicativo
